Dokumentasi API

Restful Booker API

Restful Booker API adalah API gratis dan open-source untuk tujuan pengujian dan edukasi. API ini mensimulasikan sistem booking hotel dengan endpoint untuk membuat, membaca, memperbarui, dan menghapus booking.

Base URL
https://restful-booker.herokuapp.com
Endpoint yang Diuji
Method Endpoint Deskripsi
GET /booking/:id Mengambil detail booking tertentu
PUT /booking/:id Memperbarui detail booking tertentu
