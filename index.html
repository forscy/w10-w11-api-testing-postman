API Documentation

Restful Booker API

The Restful Booker API is a free, open-source API for testing and education purposes. It simulates a hotel booking system with endpoints for creating, reading, updating, and deleting bookings.

Base URL
https://restful-booker.herokuapp.com
Tested Endpoints
Method Endpoint Description
GET /booking/:id Retrieves the details of a specific booking
PUT /booking/:id Updates the details of a specific booking
